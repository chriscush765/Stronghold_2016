##FRC Plugin
###How to install FRC plugin
1. Open Eclipse
2. Select "Help" on the top menu bar
3. Click "Install New Software"
4. Click "Add..."  at the right of "Work with" slot
5. Name: FRC Plugin
6. Location: [http://first.wpi.edu/FRC/roborio/release/eclipse/]
7. Click "OK"
8. Agree terms
9. Click "Finish"
